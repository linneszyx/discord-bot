## Run Locally

Clone the project

```bash
  git clone https://github.com/linneszyx/discord-bot.git
```

Go to the project directory

```bash
  cd discord-bot
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```
